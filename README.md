# Rog-Ally-Custom-Resolutions

Add custom resolutions for the ROG Ally.

Standard set:
- 1920x1080
- 1600x900
- 1280x720

Extended set:
- 1920x1080
- 1760x990
- 1600x900
- 1366x768
- 1280x720
- 1120x630
- 960x540

_Not all will be selectable in every game or on your desktop_

# Directions
* Run the registry file
* Reboot
